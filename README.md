# POKEDEX PROYECTO CELLS

  Este es el proyecto de la Pokedex en cells. 

1. Para iniciar este proyecto debe instalar las dependencias con:
```
npm i
```
2. seguido de esto para correrlo debe ejecutar el siguiente comando: 
```
npm run start
```
